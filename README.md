# Expense Tracker (MERN)

A full-stack Expense Tracker built with the MERN stack. Users can register, log in securely, and manage their daily expenses with a clean and responsive interface.

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Live Demo

Frontend: https://expense-tracker-mern-1-gbmm.onrender.com

Backend API: https://expense-tracker-mern-r3kt.onrender.com

## Features
- User Authentication (JWT)
- Add, Edit & Delete Expenses
- Dashboard
- Analytics
- Responsive UI
